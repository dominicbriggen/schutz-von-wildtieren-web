# Verwaltungsbereich — Bedienungsanleitung

Der Verwaltungsbereich ist unter `/admin` erreichbar (z. B.
`https://ihre-domain.ch/admin`). Er ist bewusst einfach gehalten: grosse
Schaltflächen, deutsche Bezeichnungen, keine unnötigen Einstellungen.

## Anmelden

1. Rufen Sie `/admin` auf. Sie werden automatisch zur Anmeldeseite weitergeleitet.
2. Geben Sie Ihre E-Mail-Adresse und Ihr Passwort ein.
3. Nach der Anmeldung sehen Sie die Übersicht mit allen Bereichen.

**Passwort vergessen oder erstes Passwort festlegen:** Dies läuft über die
sichere Passwort-Zurücksetzen-Funktion, die Ihnen ein Verantwortlicher bei
der Einrichtung per E-Mail zusendet. Folgen Sie dem Link in der E-Mail und
vergeben Sie dort Ihr persönliches Passwort — niemand sonst sieht dieses
Passwort.

## Abmelden

Klicken Sie in der linken Navigation (bzw. im mobilen Menü) auf «Abmelden».

## Bereiche im Überblick

| Bereich | Was Sie dort bearbeiten |
|---|---|
| **Startseite** | Hauptaussage, Vereinstext, Zitat, Startbild |
| **Verein** | Vereinspräsident, Team, Fotos, Vereinsziele |
| **Projekte** | Alle Projekte: erstellen, bearbeiten, veröffentlichen, ausblenden, löschen |
| **Neuigkeiten** | Aktuelle Meldungen: erstellen, bearbeiten, veröffentlichen, ausblenden, löschen |
| **Erfolge** | Ergebnisse je Projekt (z. B. «97 Systeme finanziert») |
| **Bilder** | Bildergalerie: Bilder hochladen, beschriften, Kategorie/Jahr zuordnen |
| **Spenden** | Spendentext, vorgeschlagene Beträge, Hinweistext |
| **Kontakt** | Adresse, Telefon, E-Mail, Instagram-Link, eingegangene Nachrichten |

## Texte bearbeiten

Öffnen Sie den gewünschten Bereich, ändern Sie die Felder und klicken Sie
auf **Speichern**. Die Änderung ist sofort auf der Website sichtbar.

Bei längeren Texten (z. B. Projektbeschreibungen): Trennen Sie Absätze mit
einer **leeren Zeile** — so werden sie auf der Website als eigene Absätze
dargestellt.

## Ein neues Projekt oder eine neue Neuigkeit erstellen

1. Öffnen Sie **Projekte** bzw. **Neuigkeiten**.
2. Klicken Sie auf **Neues Projekt** / **Neue Meldung**.
3. Füllen Sie Titel, Link-Name (nur Kleinbuchstaben, Zahlen, Bindestriche),
   Zusammenfassung, Text und ggf. Bilder aus.
4. Wählen Sie den **Status**:
   - **Entwurf** — nur für Sie sichtbar (über den «Vorschau»-Link)
   - **Veröffentlicht** — für alle Besuchenden der Website sichtbar
   - **Ausgeblendet** — vorübergehend von der Website entfernt, bleibt aber erhalten
5. Klicken Sie auf **Projekt erstellen** bzw. **Speichern**.

## Bestehende Einträge bearbeiten, ausblenden oder löschen

In der Übersicht von **Projekte** oder **Neuigkeiten** sehen Sie zu jedem
Eintrag:

- **Status-Auswahl** — direkt umschaltbar zwischen Entwurf, Veröffentlicht, Ausgeblendet
- **Stift-Symbol** — öffnet die Bearbeitung
- **Papierkorb-Symbol** — löscht den Eintrag **endgültig**, nachdem Sie eine
  Sicherheitsabfrage bestätigt haben

## Vorschau anzeigen

Beim Bearbeiten eines Projekts oder einer Neuigkeit finden Sie oben rechts
einen **Vorschau**-Link. Er öffnet die Seite auf der echten Website-Adresse
— auch dann, wenn der Beitrag noch als «Entwurf» markiert ist. Nur Sie als
angemeldete Administratorin/angemeldeter Administrator sehen diese Vorschau;
andere Besuchende sehen den Beitrag erst nach der Veröffentlichung.

## Bilder hochladen

1. Öffnen Sie den Bereich **Bilder** (für die Bildergalerie) oder nutzen Sie
   die Bild-Felder direkt in Projekten, Neuigkeiten oder beim Verein.
2. Klicken Sie auf **Bild hochladen** bzw. **Bild hinzufügen**.
3. Wählen Sie eine Bilddatei aus (JPEG, PNG, WebP oder AVIF, maximal 8 MB).
4. Ergänzen Sie bei der Bildergalerie eine Bildbeschreibung, Kategorie und
   ein Jahr, damit das Bild richtig einsortiert wird.

Bilder werden automatisch sicher gespeichert und für die Anzeige auf
Mobilgeräten optimiert — Sie müssen die Bilder nicht selbst verkleinern.

## Reihenfolge anpassen

Bei Projekten, Neuigkeiten, Erfolgen und Bildern gibt es ein Feld
**Reihenfolge**. Ein kleinerer Wert erscheint weiter oben bzw. weiter vorne.

## Erfolge pflegen

Im Bereich **Erfolge** klicken Sie auf **Neuer Erfolg**, um ein Ergebnis zu
einem Projekt zu erfassen (z. B. Zeitraum und erreichter Wert). Der
**Link-Name des zugehörigen Projekts** muss exakt dem Link-Namen des
jeweiligen Projekts entsprechen, damit das Ergebnis dort korrekt angezeigt
wird. Mit dem Schalter **Sichtbar** blenden Sie einzelne Erfolge bei Bedarf
vorübergehend aus.

## Nachrichten aus dem Kontaktformular

Im Bereich **Kontakt** sehen Sie unter «Nachrichten aus dem
Kontaktformular» alle eingegangenen Anfragen von Besuchenden der Website.
Sie können Nachrichten als gelesen markieren oder löschen.

## Löschen — immer mit Bestätigung

Jede Löschaktion (Projekt, Neuigkeit, Erfolg, Bild, Nachricht) zeigt zuerst
eine Sicherheitsabfrage. Erst nach ausdrücklicher Bestätigung wird der
Eintrag endgültig entfernt.
